You can login in using 6-digit pin.
On bookmark dashboard you can see all of your previously added bookmarks. If you would like to add new, you can do so using the right down button.
After clicking on one of your existing bookmarks, you can see details about it. Editing and deleting is possible using the representative button in each section.
In profile and settings you can see some of the stats of your bookmarks and also set the default language for OCR functionality.
